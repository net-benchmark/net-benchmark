# Security Policy

## Supported Versions

Security updates are provided for the latest released version of `net-benchmark`.  
Older versions may receive fixes at the discretion of the maintainer.

## Reporting a Vulnerability

If you discover a security issue, **do not open a public GitHub issue**.

Instead, please report it privately by emailing:

**frankosehj22@gmail.com**

Include:

- A clear description of the issue  
- Steps to reproduce (if applicable)  
- Any potential impact you have identified  

We will review your report promptly and follow up with next steps.
