# Code of Conduct

## Purpose

The net-benchmark suite (DNS, HTTP, SSL benchmarking tools) is an open-source project built on collaboration, technical excellence, and respectful communication. This Code of Conduct defines the expectations for everyone participating in the project.

## Standards of Behavior

Participants are expected to uphold the following principles:

- **Respect** — Treat others with courtesy. Disagreement is normal; disrespect is not.
- **Clarity** — Communicate clearly and constructively, especially in issues, PRs, and reviews.
- **Professionalism** — Keep discussions focused on the technical topic at hand.
- **Inclusivity** — Welcome contributors of all backgrounds, skill levels, and perspectives.
- **Integrity** — Provide honest feedback, acknowledge mistakes, and give credit where due.

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, insults, or personal attacks  
- Discriminatory language or exclusionary behavior  
- Dismissing or belittling contributors  
- Spam, trolling, or intentionally derailing discussions  
- Aggressive or hostile communication  

## Governance and Decision-Making

net-benchmark uses a BDFL governance model.

The founding maintainer, **@frankovo**, has final authority over:

- Technical direction  
- Roadmap decisions  
- Acceptance or rejection of contributions  
- Release management  

Community input is encouraged and valued, but final decisions rest with the BDFL to ensure long-term consistency and stability across the suite.

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, you may report it by opening a private issue or contacting the maintainer directly.

Reports will be reviewed promptly and confidentially.

## Enforcement

Violations of this Code of Conduct may result in actions including:

- Warning  
- Temporary or permanent ban from participation  
- Rejection of contributions  

The BDFL is responsible for interpreting and enforcing this Code of Conduct.

## Scope

This Code of Conduct applies to:

- GitHub issues, discussions, and pull requests  
- Project communication channels  
- Any interaction representing the project or its community  

## Acknowledgment

By participating in the net-benchmark suite, you agree to follow this Code of Conduct and contribute to a respectful, productive environment.
